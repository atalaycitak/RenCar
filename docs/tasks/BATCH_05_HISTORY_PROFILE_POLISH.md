# Batch 05 - Kiralamalar, Profil ve Polish

Tarih: 15-17 Temmuz 2026

## Hedef

Kiralama gecmisi, profil/ayarlar ekranlari ve gorsel polish islerini tamamlayarak app'in ana tablarini kapatmak.

## Tasklar

- [ ] Kiralamalar ekranini uygula.
- [ ] Aktif/gecmis kiralama ayrimini UI state'e ekle.
- [ ] Kiralama detay sheet veya ekranini ekle.
- [ ] Profil ekranini uygula.
- [ ] Profil durum karti: ehliyet onaylandi, hesap bilgisi, avatar.
- [ ] Ayarlar, yardim, odeme yontemleri ve cikis satirlarini ekle.
- [ ] Cikis aksiyonunu fake auth session ile bagla.
- [ ] Bottom navigation geri tusu davranisini kontrol et.
- [ ] Light/dark tema farklarini tum ekranlarda gozden gecir.
- [ ] Spacing, typography, button ve kart tutarliligini duzelt.
- [ ] Empty state'leri ekle: kiralama yok, islem yok, arac bulunamadi.
- [ ] Profil ve kiralamalar ViewModel testlerini yaz.

## Kabul Kriterleri

- Dort ana tab calisir: Harita, Cuzdan, Kiralamalar, Profil.
- Kullanici kiralama gecmisini gorebilir.
- Profil ekrani dogrulama durumunu gosterir.
- Cikis yapinca auth akisina donulur.
- Ekranlar light ve dark modda okunabilir durumdadir.
